- Conclusiones

La realización de esta práctica nos permitió comprender de forma más práctica cómo se desarrolla una aplicación en React a partir de un diseño previo realizado en Figma Make. Pudimos observar la importancia de planificar correctamente la interfaz antes de comenzar el desarrollo, organizando los componentes mediante la metodología de átomos, moléculas y organismos para facilitar la reutilización y el mantenimiento del diseño.

Además, esta práctica nos ayudó a entender mejor cómo algunos de esos átomos visuales se transforman posteriormente en componentes reales dentro de React, relacionando directamente el diseño UI con la implementación frontend. También reforzamos conceptos relacionados con la identidad visual, la organización de componentes y la experiencia de usuario, obteniendo una visión más completa del flujo de trabajo entre diseño y desarrollo web moderno.
